## Open datasets

This folder contains a set of reservoir models to play with `GeoCode` framework:
* Egg model ([source](https://doi.org/10.1002/gdj3.21)) with simulated dynamics. Unzip the `RESULTS.zip` archive before reading the model with `GeoCode`
* SPE 10 - MODEL 2 ([source](https://github.com/OPM/opm-data/tree/master/spe10model2))

For details and terms of use of the models, see the description in the model directory.

Additional models one can obtain from the `opm-data` repository https://github.com/OPM/opm-data.
